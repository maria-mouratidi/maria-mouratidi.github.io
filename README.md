# Retro Pixel Portfolio

## Overview
The Retro Pixel Portfolio is a stylish and visually engaging portfolio designed to showcase projects, skills, and experiences in a retro pixel art style. This portfolio utilizes pixelated fonts and graphics to create a nostalgic aesthetic, appealing to both tech enthusiasts and creative audiences.

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd retro-pixel-portfolio
   ```

2. **Open the Project**
   Open `index.html` in your web browser to view the portfolio.

3. **Customize Content**
   - Replace placeholder images in the `assets/images` directory with your own.
   - Update the `index.html` and other HTML files in the `pages` directory with your personal information, projects, and experiences.

4. **Add Projects**
   - Add project details in `projects.html` and include relevant images in the `assets/images/project-thumbnails` directory.

5. **Modify Styles**
   - Customize the styles in the `css/main.css`, `css/animations.css`, and `css/responsive.css` files to match your desired aesthetic.

6. **Add Interactivity**
   - Use the JavaScript files in the `js` directory to add interactivity, such as animations and audio effects.

## Features
- Retro pixelated design with a unique font.
- Responsive layout for various screen sizes.
- Sections for projects, about, and contact information.
- Sound effects for user interactions.
